# Claude Context — Learning Digital Art

This repo is a personal documentation library for mastering a digital art toolchain to produce game assets for **Beellionaire** (bee-themed slot game) and future projects.

See [README.md](README.md) for the full reference — toolchain, pipeline, hardware, file transfer workflow, and documentation index.

## Quick Context

- **Art tools:** Procreate (iPad) → Affinity Designer/Photo/Publisher (Mac + Windows) → Spine 2D (Mac + Windows)
- **Game stack:** TypeScript + Svelte/SvelteKit + Phaser 3 + `@esotericsoftware/spine-phaser`
- **Asset store:** Google Drive — AirDrop iPad→Mac, Google Drive for Desktop syncs to Windows and collaborators
- **Primary project:** Beellionaire — bee-themed slot game; use its assets as worked examples in all tutorials
- **Doc format:** All tutorials follow the template defined in README.md § Doc Format

## User

Not a seasoned digital artist — building skills alongside the tooling. Frame explanations for a learner, not an expert. Hardware: iPad + Apple Pencil, M1 Mac (primary), Windows laptop (secondary).

## What Lives Here

Markdown tutorials and reference docs covering the full pipeline from Procreate sketch to Phaser runtime. Written for someone building skills alongside the tooling — ground examples in slot game asset types (bee characters, reel frames, honeycomb textures, UI elements).
